# Sitio Valley - Presidential Simulator

Um simulador de gestão 2D com estética pixel art e tecnologia Phaser.js.

## 🚀 Como Rodar
1.  Abra o arquivo `index.html` na raiz do repositório.
2.  Para o GitHub Pages, o `index.html` deve permanecer na raiz para evitar erros 404.

## 🕹️ Mecânicas
- **Plantio**: Clique em qualquer tile de grama para plantar comida ($10).
- **Colheita**: Quando a planta amadurecer (ficar verde), clique nela para ganhar +10 de Comida.
- **Manutenção**: Contrate "Gari-Engenheiros" ($100) para consertar buracos automaticamente.
